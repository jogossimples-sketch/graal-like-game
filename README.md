# Graal-like Game
Jogo inspirado em *Graal Online West*, desenvolvido com HTML5 e Phaser.js.

## Como jogar
Acesse o jogo online via GitHub Pages: [https://jogossimples-sketch.github.io/graal-like-game/](https://jogossimples-sketch.github.io/graal-like-game/)

- Use as setas do teclado para mover o personagem
- Explore o mapa e interaja com elementos do cenário (em desenvolvimento)

## Tecnologias utilizadas
- HTML5
- JavaScript
- Phaser.js v3.60.0

## Próximas funcionalidades
- Sistema de combate
- Mapa maior com áreas temáticas
- Inventário e itens
- PvP entre jogadores

## Como rodar localmente
1. Clone o repositório: `git clone https://github.com/jogossimples-sketch/graal-like-game.git`
2. Abra o arquivo `index.html` no seu navegador
